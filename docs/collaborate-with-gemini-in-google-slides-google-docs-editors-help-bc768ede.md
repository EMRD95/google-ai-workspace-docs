# Collaborate with Gemini in Google Slides

**Source:** [Google Docs Editors Help](https://support.google.com/docs/answer/14355406?hl=en)

## Overview

Gemini in Google Slides lets you:

- Generate and edit new slides
- Generate images
- Use the style of an existing presentation
- Reference Drive files or Gmail emails
- Summarize a presentation
- Write or rewrite content

## Feature Availability

- Requires an eligible Google Workspace or Google AI plan ([Learn more](https://support.google.com/drive/answer/13952129))
- Check [supported languages](https://support.google.com/docs/answer/14925782)
- Also available in Google Workspace Experiments for personal accounts (look for a bottom bar in Slides). [How to join](https://support.google.com/docs/answer/13447104)

## Get Started

1. Open a presentation in [Google Slides](https://docs.google.com/presentation).
2. Click **Ask Gemini** (top right).
3. In the side panel, either:
   - **Select a suggested prompt**: Pick one, optionally replace example text, press Enter.
   - **Enter your own prompt**: Type in the box and press Enter.
4. **Important:** Your conversation history is temporary. It’s lost when you refresh the browser, close/reopen the presentation, or go offline. Insert suggestions to keep them.

> **Tip:** To manually clear history, click More options → **Clear history**.

## Key Tasks

### Generate Images

- **Based on slide content:** “Suggest images for this slide” or choose a suggested prompt.
- **Direct prompt:** “Generate an image of a dog with glasses.”
- Click **Insert** on the desired image.
- Edit the prompt with **Edit Prompt** or generate more with **Show more**.

### Generate a Slide

1. Enter a prompt describing the slide.
2. *(Optional)* Match another presentation’s style: click **Match a presentation style** → **Add from Drive** → select file → **Add**.
3. After preview:
   - **Insert** as a new slide (click Down arrow next to Replace → Insert)
   - **Replace** the selected slide.
- You can only generate one slide at a time, and you can edit it afterwards.

### Summarize & Write Content

- The side panel automatically shows a summary of the presentation (if text exists).
- Example prompts:
  - “Summarize this presentation in 3 sentences.”
  - “Create a slide that celebrates a colleague's work anniversary.”
- Insert with **Insert** button; regenerate with **Retry**.

### Reference Drive Files

- Use `@` mentions in prompts: e.g., “Create a slide outlining the next steps from @meeting notes and @core team sync.”
- Click **Sources** at the bottom of the response to see which files were used.

### Get Help with Slides

Ask questions like:
- “How do I start a Q&A while presenting?”
- “What is the best way to share these slides with more than 100 people?”

### Ask Questions from the Web

Include phrases like *“Use Google Search”* or *“Using web search”* in the prompt.
- “Write a paragraph describing the weather today in Mountain View, California using web search.”
- “Help me write a paragraph explaining if lightning can strike the same place twice using only information from the web.”

### Use a Gem

1. Click Ask Gemini → **Gems** (in side panel).
2. Choose a premade or custom Gem.
3. Type a question to start a conversation.

> **Tip:** Create custom Gems at [gemini.google.com](http://gemini.google.com/). Learn more about [Gems in Gemini Apps](https://support.google.com/gemini/answer/15146780).

### Edit Slides with Gemini

Describe the changes you want directly, e.g.:
- “Simplify this slide with less text and add an image.”
- “Convert to two columns.”

Then follow the same Insert/Replace options. You can close the preview or refine the prompt using Ask Gemini.

## Feedback on Generated Output

- Click **Good suggestion** or **Bad suggestion** at the bottom right of the output.
- Select which data to share (prompt, context, outputs). You can uncheck items.
- For bad suggestions, click **Next** to specify the issue and add details.
- **Don’t include personal, confidential, or sensitive info** in feedback.
