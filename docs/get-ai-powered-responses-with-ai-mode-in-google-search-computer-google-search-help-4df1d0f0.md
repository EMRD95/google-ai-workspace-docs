# 1. What AI Mode Is

**AI Mode** is Google Search’s advanced AI-powered search experience. It lets users:

- Ask broad or complex questions.
- Receive AI-generated answers.
- Ask follow-up questions.
- Explore cited web links and supporting sources.
- Go deeper than standard **AI Overviews**.

AI Mode is designed for questions that may otherwise require multiple searches, such as:

- Exploring a new concept.
- Comparing options.
- Researching multiple aspects of a topic.
- Asking follow-up questions in context.

Google says AI Mode relies on Search’s understanding of web information and high-quality web content to improve factuality. If Google does not have enough confidence in an AI-generated response, AI Mode may instead show a set of web links.

---

# 2. How AI Mode Works

AI Mode uses a technique Google calls **“query fan-out.”**

## Query Fan-Out

AI Mode:

1. Breaks a question into subtopics.
2. Searches for each subtopic simultaneously.
3. Searches across multiple data sources.
4. Combines the results into a single AI-powered response.
5. Provides links so users can explore further.

This allows AI Mode to gather more relevant web content than a single traditional search query.

# 3. Accuracy, Limitations, and Fact-Checking

Google emphasizes that AI Mode is an early-stage AI product and may make mistakes.

Possible issues include:

- Misinterpreting web content.
- Missing context.
- Producing inaccurate or incomplete responses.
- Personalization errors.

## How to Double-Check AI Mode Responses

Google recommends:

- **Check important information in more than one place.**
  - Use the AI source links.
  - Review other Google Search results.
- **Ask multiple versions of your question.**
  - Ask for different perspectives.
  - Ask for more detail.
- **Send feedback** if something appears incorrect.

# 5. Recommended Settings for Best Experience

For the best AI Mode experience, Google recommends enabling:

- **Search history**
- **Personalized recommendations**

Without these settings, users can still access AI Mode, but they cannot resume previous searches where they left off.

## Workspace Accounts

For Google Workspace users:

- If **Search and Assistant** are turned off, Search history is unavailable.
- Only a Workspace administrator can enable it through the Admin console.
