# Create Content, Images & Audio Overviews with Gemini in Google Drive — Summary

**Source:** Google Drive Help  
**Applies to:** Computer / Web version of Google Drive

> **Important**: To use Gemini in Drive, you must have an eligible Google Workspace or Google AI plan. [Learn more about Gemini features and plans](https://support.google.com/drive/answer/13952129?hl=en&co=DASHER._Family%3DBusiness-Enterprise).

---

## 1. Create a File with Gemini

You can use Gemini in Drive to generate structured content such as tables, proposals, or formatted lists, then export the result to Google Docs or Sheets.

### Steps

1. On your computer, go to [drive.google.com](https://drive.google.com/).
2. At the top right, click **Ask Gemini**.
3. Ask for a **table**, **proposal**, or **formatted list**.
4. Once generated, click either:
   - **Export to Docs**
   - **Export to Sheets**

### Important Note

> Any file created via Ask Gemini's immersive workspace automatically saves to your My Drive root folder.

---

## 2. Create Audio Overviews with Gemini

Gemini can generate an audio overview from supported files in Google Drive.

### Steps

1. On your computer, go to [drive.google.com](https://drive.google.com/).
2. Open a **PDF file**.
3. At the top right, click **Create audio overview**.
4. Wait for the audio overview to finish generating.
5. Click **Play** in the side panel to listen.
6. After generation, you’ll receive an email with the file link.

### Supported File Types

Audio overviews can be generated from:

- PDF files
- Google Docs
- Google Slides
- Microsoft Office Word Document
- Microsoft Office PowerPoint Presentation
- Plain Text files
- Videos

### Where Audio Overviews Are Saved

- All audio overviews are saved in the **“Audio overviews”** folder in **My Drive**.
- Audio files use this naming format:

```text
[File name] - audio overview - YYYY-MM-DD HH:MM UTC
```

### Important Limitation

> If you refresh the page or close the side panel or the tab before the generation is done, the file doesn’t generate.

### Audio Overview Tips & Limits

- Audio overviews can **only be generated on the web**.
- Generated audio overviews can be accessed on:
  - Web
  - iOS
  - Android
- You **can’t use a prompt in the side panel** to generate audio overviews.
- You can generate **up to 20 audio overviews per day**.

## 3. Create Images with Gemini

Gemini in Drive can also generate images from prompts.

### Steps

1. On your computer, go to [drive.google.com](https://drive.google.com/).
2. At the top right, click **Ask Gemini**.
3. Click **Tools** → **Create Images**.
4. Type a prompt.
