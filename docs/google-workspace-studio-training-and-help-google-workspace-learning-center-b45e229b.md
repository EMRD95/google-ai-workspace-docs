# Google Workspace Studio Training and Help — Summary

**Source:** [Google Workspace Learning Center](https://support.google.com/a/users/answer/16275487)  
**Purpose:** Training and support hub for **Google Workspace Studio**, a no-code automation tool for automating tasks across Google Workspace using **Gemini**.

---

## Overview

Google Workspace Studio helps users automate routine cross-Workspace tasks with Gemini. Users can:

- Tell Gemini what they want to automate
- Start from templates for common use cases
- Build a custom automation flow
- Set up and manage automations without programming

> **“Automate work tasks across Google Workspace”**

> **“Use Google Workspace Studio to automate cross-Workspace tasks with Gemini. You can tell Gemini what you want to automate, use templates for common use cases, or build a custom flow.”**
