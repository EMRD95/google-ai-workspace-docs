# Create with AI in the YouTube Create App — Summary

**Source:** YouTube Help — “Create with AI in the YouTube Create app”  
**URL:** https://support.google.com/youtube/answer/16631240

---

## Core Overview

The YouTube Create app includes experimental AI tools that help creators generate and edit content, including:

- **Veo** — generate vertical video clips from text prompts and optional reference images.
- **Nano Banana** — generate images from text prompts and optional reference images.
- **Edit with AI** — turn raw clips/photos into a polished draft video with music, voiceover, captions, and effects.

---

## Key Excerpts

> You can create and edit content with AI in the YouTube Create app.

> These AI-generated features are experimental and are currently available to a limited number of creators in select countries.

> All content on YouTube, including AI-generated content made with YouTube Create, must follow our Community Guidelines.

> We may not accept prompts that violate our policies or touch on sensitive topics.

> Labels clearly indicate to viewers that the content they’re watching is synthetic.

# 1. Generate a Video with Veo

## What Veo Does

**Veo in YouTube Create** lets creators generate vertical AI video clips for shots they did not capture themselves. It supports control over:

- **Lighting**
- **Style**
- **Audio / sound effects via prompt**
- **Portrait video format**

## Availability & Requirements

> Veo on YouTube Create is currently available on mobile devices in Australia, Canada, India, New Zealand, and the United States.

Key requirements:

- Available only on **mobile devices**
- Available in:
  - Australia
  - Canada
  - India
  - New Zealand
  - United States
- **Prompts must be in English**
- Veo is currently available only for certain AI-generative tools
- Prompt limit: **900 characters**
- Reference images: up to **3 images**
- Aspect ratio: **9:16 Portrait**, default

## How to Access Veo

Open the **YouTube Create app**, then access the feature from one of several places:

1. **Homepage banner**
2. In the editor, tap **Add** to add a new video to the timeline, then choose the option to generate a new clip
3. In the editor, choose the option to generate a new clip when adding an overlay
4. From the bottom navigation tool in the editor, tap **Generate video**

## How to Generate a Video

1. Open the YouTube Create app.
2. Access **Generate video** through one of the available entry points.
3. Enter a text prompt describing the desired video.
   - You can also tap a suggested prompt.
   - Prompt limit: **900 characters**
   - Optional: upload up to **3 reference images** from your gallery.
4. Customize the prompt with:
   - **Style:** choose from **11 predefined styles**, such as:
     - Photorealistic
     - Vintage Film
     - Claymation  
     **Photorealistic** is the default.
   - **Lighting:** choose options like:
     - Soft morning light
     - Golden hour
     - Blue hour
     - Moonlight
   - **Audio:** describe desired audio and sound effects in the prompt.
   - **Aspect Ratio:** **9:16 Portrait**, default.
5. Tap **Generate**.
6. Tap **Use Video** to add the generated clip to your timeline.

# 2. Generate an Image with Nano Banana

## What Nano Banana Does

Nano Banana lets creators generate new images using:

- Text prompts
- Optional reference images from the device gallery

## Availability & Requirements

> This feature is currently available to users over 18 on mobile devices in Australia, Canada, India, New Zealand, and the United States. Prompts must be in English or Hindi.

Key requirements:

- Users must be **over 18**
- Available only on **mobile devices**
- Available in:
  - Australia
  - Canada
  - India
  - New Zealand
  - United States
- Prompts must be in:
  - **English**
  - **Hindi**
- Prompt limit: **900 characters**
- Reference images: up to **3 images**
- Generates **4 image options** per prompt

## How to Access Nano Banana

Open the **YouTube Create app**, then access image generation from:

1. **Homepage banner**
2. In the editor, tap **Add** to add a new video to the timeline, then choose the option to generate a new image
3. In the editor, choose the option to generate a new image when adding an overlay
4. From the bottom navigation tool in the editor, tap **Generate image**

## How to Generate an Image

1. Open the YouTube Create app.
2. Access **Generate image** through one of the available entry points.
3. Enter a description of the image you want to create.
   - Prompt limit: **900 characters**
   - Optional: add up to **3 reference images** from your device gallery.
4. Tap **Generate**.
   - The app creates **4 images** based on your prompt.
   - You can provide feedback with **thumbs up** or **thumbs down**.
5. Select the image you like.
6. Choose one of the following:
   - **Use Image** — adds the image to your timeline.
   - **Create Video** — turns the image into an **8-second video**.
