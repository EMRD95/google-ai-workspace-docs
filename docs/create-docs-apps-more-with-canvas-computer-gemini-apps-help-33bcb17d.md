# Create docs, apps & more with Canvas — Gemini Apps Help

**Source:** https://support.google.com/gemini/answer/16047321  
**Platform covered:** Computer  
**Product:** Gemini Apps / Canvas

---

## Overview

Canvas in Gemini lets you collaborate with Gemini to **create or edit docs, apps, slides, code, quizzes, Audio Overviews, infographics, web pages, and other formats**.

> **Important:** To use this feature, you must be signed in to Gemini Apps.

Canvas supports:
- Creating new docs, slide decks, quizzes, apps, and other content
- Editing generated docs and apps
- Direct code editing
- Adding Gemini-powered AI features to apps
- Exporting to Google Docs, Google Slides, PDFs, Colab, or copying content
- Sharing Canvas apps/content through public links
- Copying and editing shared Canvas content, with age and platform restrictions

---

## Start a New Chat with Canvas

1. Go to [gemini.google.com](https://gemini.google.com/).
2. Below the text box, click **Add Files** → **Canvas**.
3. Enter your prompt in the text box.
   - **Tip:** If you want a specific format or document type, such as a slide deck, include that in your prompt.
4. Optional: Click **Add files** to upload files or an image with your prompt.
5. Click **Submit**.

### Example prompts

> Create a slide presentation about…

> Create a quiz to test me on…

## Apps in Canvas

### Edit Apps & Code

You can edit an app either by:
- Asking Gemini to make changes using the prompt box
- Directly editing the app’s code

**Changes in Canvas are auto-saved.**

#### Open code, errors, logs, and changes

- To open and edit code: click **Code** in the top-right of the Canvas panel.
- To view preview errors and logs: click **Show console**.
- To view recent code changes: click **Code** → **Show recent changes**.

---

### Ask Gemini to Edit Selected Parts of an App

1. Open the chat with the app you want to edit.
2. In the bottom-right of the Canvas panel, click **Select & ask**.
3. Use the cursor to select the section you want to edit.
4. Enter a prompt describing the update you want.
5. Press **Enter**.

---

### Add AI to an App

> **Important:** To add AI to your app, you must be signed in to Gemini Apps and be **18 or over**.

You can add Gemini-powered features such as:
- Text generation
- Image generation

Steps:

1. Open the chat with the app you want to edit.
2. In the bottom-right of the Canvas panel, click **Add Gemini features**.
3. Gemini will provide an overview of the AI feature added to the app.
4. You can reply in the text box and ask Gemini to make additional changes.

## Docs in Canvas

### Edit Docs in Canvas

You can edit docs directly in Canvas or ask Gemini to help via the prompt box.

**Changes are auto-saved.**

#### Ask Gemini to edit selected text

1. Open the chat you want to edit.
2. In the Canvas panel, select the text you want to edit.
3. Enter a prompt explaining what you want updated.
4. Press **Enter**.

**Tip:** After selecting text, you can also click **Suggest edits** to have Gemini recommend changes.

---

### Ask Gemini to Make Quick Edits

1. Open the chat you want to edit.
2. In the Canvas panel, choose a quick-edit option:
   - **Change length** — adjust the length of text.
   - **Change tone** — make the text more casual or formal.

---

### Edit Text Style & Formatting in a Doc

1. Open the chat you want to edit.
2. Select the text in the Canvas panel.
3. Use the toolbar to format the text.

Available formatting options include:
- Stylize headings
- **Bold text**
- *Italicize text*
- Create bulleted lists
- Create numbered lists

**Tips:**
- If the full toolbar is not visible, click **More**.
- To view saved versions, click **Previous Version** or **Next Version** at the top.

## Add LaTeX to a Canvas Document

LaTeX is described as:

> LaTeX is a document preparation system widely used in academic, scientific, and technical fields to create high-quality documents with complex formatting, such as mathematical equations, tables, and multi-column layouts.

### Add LaTeX

1. Open the chat with the doc you want to edit.
2. At the top-right of the Canvas panel, click **Create**.
3. Select **LaTeX**.
4. Click on the formula within Canvas.

### Exporting or Sharing Docs with LaTeX

- You can export a doc containing LaTeX as a **PDF**, with preview before download.
- You can copy any Gemini response that includes LaTeX; the copied content will include the **unrendered LaTeX code**.

## Create Something Based on a Canvas Doc

> **Important:** For now, this feature isn’t available to users under 18.

You can transform an existing Canvas doc into other formats.

1. Open the chat with the doc you want to edit.
2. In the top-right of the Canvas panel, click **Create**.
3. Choose what you want to create.

Available options:
- **Slide show, custom visual, or app**  
  Enter a prompt describing what you want to turn the report or doc into.
- **Audio Overview**
- **Quiz**
- **Infographic**
- **Web page**
