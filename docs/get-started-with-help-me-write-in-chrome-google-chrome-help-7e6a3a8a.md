# Get started with **Help me write** in Chrome — Summary

**Source:** Google Chrome Help — <https://support.google.com/chrome/answer/14582048>  
**Purpose:** Explains how to use Chrome’s **Help me write** feature, requirements, troubleshooting, privacy/data use, and limitations.

---

## Overview

**Help me write** is an experimental Chrome feature that provides AI-powered writing suggestions in open text fields on the web. It is intended for **short-form writing**, such as:

- Reviews
- Surveys
- Replies
- Drafting or refining text in web forms

It can generate new text from a prompt or rewrite selected text.

---

## Key Requirements

To use **Help me write**, you must:

- Be **located in the US**
- Be **at least 18 years old**
- Have **English** set as your Chrome browser language
- Be **signed in** to your Google Account in Chrome
- Have **Make searches and browsing better** turned on

Availability:

- Language: **English only**
- Platforms: **Windows, Mac, and Linux**
- Region: **US only**

---

## How to Get Writing Suggestions

1. Open **Chrome** on your computer.
2. Go to a website.
3. Right-click an open text field.
   - To rewrite text, **highlight the text first**, then right-click it.
4. Select **Help me write**.
5. In the **“Help me write”** box, enter your prompt.
   - A prompt can be:
     - A phrase
     - A question
     - An instruction
     - A sentence
   - If you highlighted text, it will automatically appear in the box.
6. Choose a writing goal:
   - **Polish:** Helps rewrite text and add noticeable value without changing the user’s tone.
     - This is the default option.
   - **Elaborate:** Creates a more detailed or creative draft with additional sentence structures and details.
   - **Formalize:** Rewrites with a more professional tone.
7. Select **Create**.
8. Select **Insert** or **Replace** to add the suggestion to the text field.

---

## Editing and Refining Suggestions

After generating a suggestion, you can:

- Edit your original input:
  - Select **Edit**, then **Update**
- Edit directly in the generated response:
  - Click anywhere in the response box
- Modify the response using options such as:
  - **Casual**
  - **Formal**
  - **Elaborate**
  - **Shorten**
  - **Rephrase**
  - **Retry**
- Use **Undo** or **Redo** for changes to generated text

You can also rate suggestions:

- Select **Thumbs Up**
- Select **Thumbs Down**
  - If you choose Thumbs Down, you can explain why the suggestion didn’t meet your needs.

## Troubleshooting

### Help me write not appearing in the context menu

If **Help me write** does not appear, you may be:

- Signed out of Chrome
- Missing the required setting: **Make searches and browsing better**

To check:

1. Open Chrome.
2. Go to:

```text
chrome://settings/account
```

3. Confirm that **“Make searches and browsing better”** is turned on.

---

### “Not enough text” error

You can’t get suggestions if your prompt or phrase is fewer than **3 words**.

Use at least **3 words**, such as:

- A few words summarizing your thoughts
- A first draft
- An instruction, for example:

> “Reply yes to this wedding invite with excitement”

---

### Unsupported language or unintelligible prompts

You’ll receive an error if the input is:

- In an unsupported language
- Unintelligible

---

### Sensitive information error

You may receive an error if your input contains sensitive information.

Avoid entering:

- Medical details
- Financial details
- Personal information

Google also advises not using Help me write on sites that contain personal or sensitive information.

---

## Important Warnings and Limitations

**Important:**

- Help me write **doesn’t provide medical, legal, or financial advice**.
- Do not include personal information such as:
  - Name
  - Phone number
  - Address
  - Social Security number
  - Credit card information
- Do not use the tool on sites containing personal or sensitive information.

If you do enter personal or sensitive information:

> Chrome won’t use it for model training purposes.

Because Help me write uses generative AI and is experimental:

- Suggestions may be **inaccurate**
- Suggestions may be **offensive**
- You should always review suggestions before accepting them

Help me write does **not** submit text for you. When inserted, the suggestion only appears in the text box. You must decide whether to:

- Submit it
- Edit it
- Delete it
- Close the webpage

---

## Data Use and Privacy

When you use Help me write, the following may be sent to Google:

- Your text
- The content of the page where you’re writing
- The page URL

Google uses this information to:

- Improve the feature
- Support generative model research
- Support machine learning technologies
- Provide more contextual writing suggestions

Google may also use feedback to improve safety policies and address challenges with large language models.
