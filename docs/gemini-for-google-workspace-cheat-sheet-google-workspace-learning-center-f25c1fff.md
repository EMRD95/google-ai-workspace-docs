# Overview

Gemini for Google Workspace can help users:

- Generate new written content in **Google Docs**
- Rewrite, summarize, shorten, or ask questions about selected text in **Google Docs**
- Draft and refine emails in **Gmail**
- Create organized tables in **Google Sheets**

Availability depends on whether the user’s organization supports Gemini features.

# Write with Gemini in Google Docs

Gemini in Docs can help users either write new text or work with existing selected text.

---

## Write Something New in Google Docs

Use this when you want Gemini to generate new text at a specific location in a document.

### Steps

1. Click in the document where you want to insert new text.
2. Click **Help me write**.
3. Enter a prompt describing what you want Gemini to write.
4. Click **Create** to preview the suggested text.
5. Optionally insert the generated text into the document.

### Key Action

**Prompt Gemini clearly** with what you want written, then review the preview before inserting.

### Related Help

- [Google Docs Editors Help Center — Help me write in Docs](https://support.google.com/docs/answer/13447609)

---

## Rewrite or Ask About Existing Text in Google Docs

Use this when you already have text and want Gemini to modify or analyze it.

### Steps

1. Select the text you want to rewrite or ask Gemini about.
2. Click **Help me write**.
3. Choose an option such as:
   - **Summarize**
   - **Shorten**
4. Enter a prompt asking Gemini to write something based on the selected text.

### Common Uses

- Summarizing selected content
- Shortening text
- Rewriting for tone, clarity, or length
- Asking Gemini to generate new content based on a passage

# Draft Email with Gemini in Gmail

Gemini can help create new email drafts or rewrite selected email text in Gmail.

## How to Use Gemini in Gmail Compose

Use Gemini inside a Gmail compose window.

### Steps

1. In a **Compose** window, click **Help me write**.
2. Enter your prompt.
3. Click **Create**.
4. Click **Recreate** to generate a new suggestion if needed.
5. Click **Refine** to:
   - Rewrite the suggested text
   - Choose a refinement option
   - Change your prompt
6. Click **Insert** to add the suggestion to your email.

### Key Controls

- **Create** — Generates an email draft or suggestion
- **Recreate** — Produces a different version
- **Refine** — Adjusts or rewrites the suggested text
- **Insert** — Adds the final suggestion into the email

### Related Help

- [Gmail Help Center — Help me write in Gmail](https://support.google.com/mail/answer/13384326)

# Create Tables with Gemini in Google Sheets

Gemini can generate structured tables in a blank Google Sheet based on a prompt.

## How to Create a Table

Start with a blank sheet, then ask Gemini to organize information into a table.

### Steps

1. At the top of the sheet, click **Ask Gemini**.
2. Select a suggested prompt or create your own.
3. Press **Enter**.
4. Click **Insert** to add the generated table to your sheet.
