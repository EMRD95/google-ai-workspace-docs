# Google Workspace Studio Cheat Sheet — Summary

**Source:** Google Workspace Learning Center  
**Article:** *Google Workspace Studio cheat sheet*  
**URL:** https://support.google.com/a/users/answer/16430708

## Purpose

Google Workspace Studio lets users create **cross-Workspace service automations** directly in a browser, such as automations for:

- Email management
- Project tracking
- Other multi-app Workspace workflows

## Key Excerpt

> Create cross-Workspace service automations for email management, project tracking, and more, right in your browser.

## Open Workspace Studio

Use this link to open Workspace Studio:

```text
https://studio.workspace.google.com
```

# Main Workspace Studio Areas

## 1. Discover Page

The **Discover** page is where users can browse templates, create new flows, and access existing flows.

### Available Actions

1. **Create a flow using AI**
2. **Filter templates by template type**
3. **Create a flow from scratch**
4. **Return to the Discover page**
5. **Open a list of your flows**

### Summary

The Discover page acts as the starting point for building automations. Users can either use AI assistance, start from a template, or build a flow manually from scratch.

---

## 2. Build and Edit a Flow

The flow editor is where users configure, manage, and activate automations.

### Available Actions

1. **Return to your list of flows**
2. **Rename the flow**
3. **View flow activity**
4. **Share the flow**
5. **Copy or delete the flow**
6. **Add a variable**
7. **Delete or move a step**
8. **Add a step**
9. **Turn on the flow**

### Summary

In the flow editor, users can modify the structure and behavior of a flow by adding steps, variables, and managing existing steps. They can also rename, share, duplicate, delete, and enable the flow.

---

## 3. Review Flow Activity

The activity view helps users monitor flow runs and troubleshoot execution status.

### Available Actions and Details

1. **Filter to see only runs that are completed, in progress, or had errors**
2. **Refresh to see the latest activity**
3. **Collapse activity for a run**
4. **Open the flow for editing**
5. **View the time the run started**
6. **Expand activity for a run**

### Summary

The flow activity page provides visibility into automation runs. Users can filter by run status, inspect activity details, refresh results, and open the flow for edits.
