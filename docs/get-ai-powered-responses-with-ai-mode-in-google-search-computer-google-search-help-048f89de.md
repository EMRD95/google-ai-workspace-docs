# 1. What AI Mode Is

**AI Mode** is an AI-powered Google Search experience designed for deeper, more interactive search.

It can:

- Answer complex questions using AI-generated responses.
- Let users ask **follow-up questions**.
- Provide **helpful web links** for deeper exploration.
- Search across multiple subtopics at once using a technique Google calls **“query fan-out.”**
- Build on the capabilities of **AI Overviews**, with more advanced reasoning and interaction.

Google says AI Mode relies on Search’s understanding of web information and is supported by high-quality web content to improve factuality. If Google does not have enough confidence in an AI-generated response, AI Mode may instead show a set of web links.

---

# 2. Important Limitations

AI Mode is an early-stage AI product and may make mistakes.

Possible issues include:

- Misinterpreting web content.
- Missing important context.
- Producing inaccurate or unhelpful answers.
- Getting personalization wrong.

Google recommends users verify important information using multiple sources.

# 4. Recommended Settings for Best Experience

For the best experience, Google recommends enabling:

- **Search history**
- **Personalized recommendations**

Without these settings:

- You can still access AI Mode.
- You may not be able to pick up where you left off with previous searches.

Special cases:

- **Workspace users** with Search and Assistant turned off cannot use Search history unless enabled by an administrator in the Admin console.
- **Supervised children’s accounts** can have settings managed through the **Family Link app**.

# 5. How to Access AI Mode

You can access AI Mode in three ways:

1. Go to:  
   **https://google.com/ai**

2. Go to:  
   **https://www.google.com**  
   Enter a question in the Search bar, then select **AI Mode**.

3. In the **Google app**, tap **AI Mode** on the home screen.

---

# 6. How to Use AI Mode

AI Mode supports multiple input types:

- Text
- Voice
- Images
- PDFs

It also supports follow-up questions and continued exploration.

## Use Text

1. Enter a question in the **Ask anything** bar near the bottom of the screen.
2. To ask a follow-up or new question, type again into the **Ask anything** bar.

## Use Voice

1. Tap the **Microphone** icon in the **Ask anything** bar.
2. Say your question.
3. Tap the microphone again to ask a follow-up or new question.

## Use an Image or PDF

1. Click the image upload icon to upload an image, or the PDF upload icon to upload a PDF from your computer.
2. You can also upload by:
   - Drag and drop
   - Copy and paste
3. Ask a follow-up or new question in the **Ask anything** bar.
