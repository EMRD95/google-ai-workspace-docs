# Collaborate with Gemini in Google Drive (Workspace Experiments)

**Source:** [Google Drive Help](https://support.google.com/drive/answer/14217860?hl=en)

---

## Feature Availability
- Requires an eligible **Google Workspace** or **Google AI plan** (see [Gemini features and plans](https://support.google.com/drive/answer/13952129))
- Also available through **Google Workspace Experiments** (trusted tester program for personal accounts)
  - Check access: open a new Google Slides presentation; if you see a bottom bar, you have access. [Get started with Workspace Experiments](https://support.google.com/docs/answer/13447104).

## Supported Languages
- Gemini in Drive side panel supports a subset of languages. [Full list](https://support.google.com/drive/answer/13607340).

## What You Can Do with Gemini in Drive
- Summarize a long document or an entire folder
- Discuss a topic by synthesizing info from one or more files (across Drive or within a specific folder)
- Retrieve quick facts about a project without opening documents
- Move or create Google files and folders

---

## Get Started with Gemini in Drive

1. Go to [drive.google.com](https://drive.google.com/)
2. Click **Ask Gemini** (star icon) at the top right
3. In the side panel, choose a suggested prompt or write your own
4. Optional: to clear history, click **More options** → **Clear history**

> **Important:** Conversation history is lost when you reload the browser, close/reopen the document, or go offline.

### Prompt Examples
- *Select a suggestion* – pick from the list; choose **More suggestions** for more; replace placeholder text and press **Enter**
- *Enter your own* – type a prompt and press **Enter**

## Things You Can Ask Gemini

### Search for Files
- Click **Ask Gemini** and describe the file by content (e.g., “Find the spreadsheet with Q3 revenue”)

### Ask Questions About a Specific File
- Click **Ask Gemini**, then use **Add Sources** or type **@** + file name
  - Example: `"Summarize the key points in @Sales Team Meeting Notes"`
- After response: click **Sources** to see referenced files; use **View more**, **Retry** for new text.

> **Tip:** Remove selected sources with **Clear selected sources**.

Other file-level prompts:
- “When is our contract with Vendor A expiring?”
- “Summarize this file for me in bullet points”
- “How many customer sign-ups did we get for this test application?”
- “Who wrote the Project B design doc?”

### Ask Questions About a Specific Folder
- Click **Ask Gemini**, add folder with `@` + folder name
  - Example: `"Find my files for Project Clover in @2024 Projects"`
- After response, **Sources** shows which files were used.

Folder-level prompts:
- “When is the next product review based on this folder?”
- “Tell me about @[Contract folder]”
- “What are the documents from Alice in this folder?”

### Summarize a File or Folder
- Use `@` to specify the file/folder, optionally combine with a summarize prompt.
  - Example: `"Summarize the key points in @Sales Team Meeting Notes"`
- For summaries, **Sources** indicates files used; use **Retry** for new version.

### Summarize Information Across Multiple Files
- Ask broad questions without `@` to search across all accessible files.
  - Examples:
    - “Help me understand our team’s 2024 strategy.”
    - “Summarize project X.”
- Response includes **Sources**, feedback buttons, and **Retry**.

### Move Files
- Use `@` to mention up to **5 files** and a target folder.
  - Examples:
    - “Move @Acme-Results to @Clients-Acme folder.”
    - “Move @Acme-Drafts and @Acme-Results to @Clients-Acme folder.”
- Press Enter, then click **Move** in the preview card. Permissions may change.

### Create New Files & Folders
- Prompt to create folders or files (if no name given, Gemini prompts for one)
  - “Create a folder called Design.”
  - “Create a sheet called Acme Campaign Planning.”
- Click **Create** in the preview card; new item is placed in the current folder (if open with edit access) or in My Drive.
