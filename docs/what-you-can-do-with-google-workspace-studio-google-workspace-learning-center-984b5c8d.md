# What You Can Do with Google Workspace Studio — Summary

**Source:** Google Workspace Learning Center  
**Article:** “What you can do with Google Workspace Studio”  
**URL:** https://support.google.com/a/users/answer/16430812

---

## Core Description

> **Google Workspace Studio is an online app that lets you automate your everyday, routine tasks across Google Workspace services with the help of Gemini—no programming required.**

Google Workspace Studio helps users create automated workflows, called **flows**, across Google Workspace apps. It is designed for non-programmers and uses **Gemini** to assist with automation.

---

## Main Use Case

Google Workspace Studio is for automating **everyday, routine tasks** across Google Workspace services.

Key points:

- Works online through a web app.
- Uses **Gemini** to help create automations.
- Does **not** require programming.
- Automations are referred to as **flows**.
- Intended for use with a Google Workspace account.

## What You’ll Need

To use Google Workspace Studio, you need:

| Requirement | Details |
|---|---|
| **An idea for a flow** | You should have an idea for a useful automation or workflow. |
| **Google Workspace access** | You need access to Workspace Studio through a **Google Workspace account**. If you don’t have access, contact your administrator. |
| **Supported computer/browser** | You need a computer with a supported web browser. |
| **No mobile support currently** | Workspace Studio **isn't available on mobile devices at this time**. |
| **Gemini access for AI features** | To use generative AI features, your Google Account must be allowed access to **Gemini and its features**. |

---

## Important Access Notes

- Workspace Studio requires a **Google Workspace account**.
- Access may depend on your organization’s settings.
- If Workspace Studio is unavailable, users should contact their Google Workspace administrator.
- Generative AI features require the account to be permitted to use **Gemini**.

Relevant help link:

- **Contact your administrator:** https://support.google.com/a/users/answer/6208960

## Device and Browser Requirements

Workspace Studio is currently intended for desktop use.
