# Customize Chrome Themes with AI — Summary

**Source:** [Google Chrome Help](https://support.google.com/chrome/answer/14516713)  
**Feature:** Create custom Chrome themes using Generative AI via **Customize Chrome → Create with AI**.

---

## Overview

Chrome lets users create custom themes with Generative AI. Users can choose a **subject**, **style**, **mood**, and **color** to generate a theme for the Chrome New Tab page.

> “Create your own theme for Chrome through Generative AI. You can choose a subject, style, mood, and color based on your preferences with the Create with AI feature under ‘Customize Chrome.’”

---

## Requirements

To use **Create with AI**, you must be:

- Located in the **United States**
- At least **18 years old**
- Signed in to your **Google Account** in Chrome

---

## How to Create a Chrome Theme with AI

1. On your computer, open **Chrome**.
2. Open a **New Tab** page.
3. At the bottom right, select **Customize Chrome**.
4. Select **Change theme** → **Create with AI**.
5. Under **“Create theme with AI”**, configure the theme:
   - **Choose a subject:** Pick from a list of popular subjects.
   - Under **“Add optional details”**:
     - **Select a style:** Choose a preferred style. Styles include thumbnail previews.
     - **Select a mood:** Pick a mood from the list.
     - **Select a color:** Choose from available color options, or use **Custom color** for more choices.
6. Select **Create**.
7. Chrome generates a set of AI-created themes based on your selections.
8. Select a generated theme to apply it to the **New Tab** page.

**Tip:** Theme creation may take up to **10 seconds**.
