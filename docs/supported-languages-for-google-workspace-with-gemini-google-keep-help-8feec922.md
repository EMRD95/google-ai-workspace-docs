# Supported Languages for Google Workspace with Gemini — Summary

**Source:** Google Keep Help — “Supported languages for Google Workspace with Gemini”  
**URL:** https://support.google.com/keep/answer/14925782?hl=en

## Key Excerpt

> Some features in Google Workspace with Gemini are available in multiple languages.

> For other Google Workspace with Gemini features, English is the only supported language. To use Google Workspace with Gemini features that are supported in English only, set your Google Account language to English.

### Help me write

For eligible **Google Workspace** or **Google AI** plans, Gemini can draft emails in the **8-language Gemini set**:
