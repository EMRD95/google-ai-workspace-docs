# Use the Gemini web app to get answers in Chrome

You can now use Chrome’s site shortcut to get direct access to the Gemini web app.

## What you need

To use the Gemini web app, you need to:

- Have a personal Google Account that you manage on your own. You still can’t access the Gemini web app with a Google Account managed by Family Link. [Read the Gemini Apps Privacy Notice](https://support.google.com/gemini/answer/13594961#privacy_notice).
- Be 13 (or [the applicable age in your country](https://support.google.com/accounts/answer/1350409)) or over.

Additional availability information

- [Learn about supported languages and countries for the Gemini web app](https://support.google.com/gemini/answer/13575153).
- [Learn about supported languages and countries for the Gemini mobile app](https://support.google.com/gemini/answer/13575153).
- [Learn more about how to use Gemini Apps on your work or school Google Account](https://support.google.com/gemini/answer/14620100).
- To use [gemini.google.com](https://gemini.google.com/), you need access to a supported browser: Chrome, Safari, Firefox, Opera, or Edgium.

## Use the Gemini web app in Chrome

**Important:** You can’t edit, delete, or set Gemini as your default search engine.

1. On your computer, open Chrome.
2. In the address bar, enter `@gemini`.
3. Press **Tab** or **Space**.

   - You can also select **Chat with Gemini** in the suggestions.
4. Enter your prompt.
5. Press **Enter**.

   - To get the response, you’ll be redirected to [gemini.google.com](http://gemini.google.com/).

**Tip:** You can deactivate Gemini site shortcut in Settings. [Learn to manage search engines and site shortcuts](https://support.google.com/chrome/answer/95426#edit).
