## Core Purpose

This guide shows marketing-related ways to use **Gemini for Google Workspace** inside **Google Docs, Gmail, Sheets, and Slides** to draft, rewrite, organize, summarize, and create marketing assets faster.

Primary use cases covered:

- Generate blog post and social media ideas
- Draft corporate profiles
- Convert content into press releases
- Write company-wide announcement emails
- Organize marketing campaigns in Sheets
- Outline presentations
- Generate custom visuals in Slides
- Create executive one-page summaries

## 1. Generate Ideas for Blog Posts

### What Gemini Can Help With

Use Gemini in **Google Docs** to generate a blog post or social media content around a topic such as:

- A new product
- A webinar
- A specific customer segment
- A specific industry

Gemini is intended to help users:

- Get started
- Generate ideas
- Develop and evolve a pitch
- Adjust tone through revisions

### Setup

Use **Help me Write** in a **Google Doc** to write new text.

### Key Prompt Examples

```text
"Write a blog post promoting our webinar on [ topic] targeting [ audience]."
```

Example values:

- `topic`: “classroom management”
- `audience`: “teachers”

```text
"Write 3 social media posts targeted at [ audience] telling them about our new product."
```

Example audiences:

- “tech enthusiasts”
- “home owners”

## 2. Draft Corporate Profiles

### What Gemini Can Help With

When a new executive joins the company, Gemini can help quickly draft a corporate profile for the company website.

It can draft a profile based on:

- The executive’s job role
- Optional personal information, such as a hobby or public accomplishment

The guide notes that Gemini may create **placeholder information** that should be reviewed, replaced, and refined.

### Setup

Use **Help me Write** in a **Google Doc** to write new text.

### Key Prompt Example

```text
"Help me write a corporate profile for our company web site about a new [ job role] joining the company. Mention [ something personal about the person]."
```

Example values:

- `job role`: “CFO” or “VP of Marketing”
- `something personal`: a hobby or past accomplishment from the person’s public job profile

## 3. Rewrite Content as a Press Release

### What Gemini Can Help With

After creating a blog post, corporate profile, or other public announcement content, Gemini can help transform it into a **press release**.

Gemini can provide:

- A typical press release framework
- Placeholder information
- A draft to refine before publishing

### Setup

In a **Google Doc**:

1. Insert the information you want to summarize.
2. Select that information.
3. Use **Help me Write** to ask Gemini about the selected text.

### Key Prompt Example

```text
"Rewrite this corporate profile as a press release announcing that this executive is joining our company."
```

## 4. Email a Company Announcement

### What Gemini Can Help With

Gemini can help draft internal announcement emails for the entire company.

Users should provide:

- A brief description of the announcement
- Desired tone

The Gemini output should be used as a **framework** and revised as needed.

### Setup

Use **Help me Write** in a **Gmail compose window** to write a new email.

### Key Prompt Example

```text
"Email the entire company notifying them that we acquired [ company name]. Use an excited tone."
```
