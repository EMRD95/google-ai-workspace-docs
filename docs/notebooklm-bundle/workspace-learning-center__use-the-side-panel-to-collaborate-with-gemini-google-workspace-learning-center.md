## What Is the Gemini Side Panel?

The **Gemini side panel** lets users chat with Gemini while working inside Google Workspace apps.

It can assist with:

- Summarizing long email threads and suggesting responses.
- Creating and refining text using the context of the current document.
- Summarizing spreadsheets and creating tables or formulas.
- Generating slides, creating images, and summarizing presentations.
- Answering questions about files stored in Drive.

The key benefit is that Gemini can use context from emails, documents, files, and other Workspace content **without requiring users to switch applications or tabs**.

# Using the Side Panel in Gmail

Gemini in Gmail can summarize emails, identify action items, search for information, and locate specific messages.

## Sample Prompts in Gmail

### Summarize an email thread

> "Summarize this email."

> "Create a list of action items for me based on this email."

### Find information from previous emails and Drive files

> "What time is my next flight?"

> "What are the main points of the weekly team meeting notes from my documents?"

### Search for certain types of emails

> "Show my unread read emails."

> "Find emails from \[person\] sent last week."

**More help:** [Gmail Help Center](https://support.google.com/mail/answer/14355636)

# Using the Side Panel in Docs

Gemini in Docs can write, refine, summarize, answer questions about Drive/Gmail content, and help with Docs features.

## Sample Prompts in Docs

### Write and refine text in context

> "Write an announcement email about a new feature."

> "Make the announcement more fun."

### Summarize Drive files and Gmail emails

> "Catch me up on the latest Monthly Review emails."

> "What are the main points of @Meeting notes: Core team sync?"

### Get help with Google Docs

> "How do I make my document pageless?"

> "Can I add markups to my document?"

**More help:** [Docs Editors Help Center](https://support.google.com/docs/answer/14355406)

# Using the Side Panel in Sheets

Gemini in Sheets can help create formulas, generate tables, summarize spreadsheets, and explain spreadsheet features.

## Sample Prompts in Sheets

### Create formulas

> "Create a formula to find cell C1 in range D:G, output value in column G."

> "Create a formula to find the number of unique customers from North America region."

> "Summarize this spreadsheet."

### Get help with Google Sheets

> “How can I remove duplicates?”
