# Collaborate with Gemini in Google Sheets — Summary

## Overview

Gemini in Google Sheets lets eligible users collaborate with AI directly inside spreadsheets. It can help create and edit spreadsheet content, analyze data, generate charts, summarize connected Workspace content, and perform spreadsheet actions through natural-language prompts.

### Gemini in Sheets can:

- Create tables.
- Create formulas.
- Generate data analysis and insights.
- Build charts and graphs.
- Summarize emails and files from Drive and Gmail.
- Perform spreadsheet actions, including:
  - Apply conditional formatting.
  - Create a pivot table.
  - Add a dropdown or checkbox.
  - Sort, apply, or clear a filter.
  - Find and replace text.
  - Set a number format.
  - Insert, delete, or freeze rows and columns.
  - Fill a range.
  - Format a table.
  - Solve optimization tasks.

---

## Feature Availability & Requirements

- Requires an eligible **Google Workspace** or **Google AI** plan.
- Supported languages vary by Gemini feature.
- Gemini works best with **native Google Sheets files**.

> **Important:** This feature works best with native Google Sheets files. To use Gemini features in an Excel file (.xlsx), select **File** → **Save as Google Sheets**.

## Getting Started with Gemini in Sheets

### Open Gemini

1. Open a spreadsheet in [Google Sheets](https://docs.google.com/spreadsheets/u/0/).
2. Click **Ask Gemini** in the top-right corner.
3. In the side panel:
   - Select a suggested prompt, or
   - Enter your own prompt.
4. Press **Enter**.

### Using Suggested Prompts

1. Select a suggestion from the list.
2. Click **More suggestions** for additional options.
3. Replace example text in the prompt box with your own request.
4. Press **Enter**.

### Entering Your Own Prompt

1. Type a prompt in the Gemini prompt box.
2. Press **Enter**.

### Optional Controls

- **Undo:** Reverse a Gemini action.
- **Clear history:** Remove recent Gemini side-panel history.
- **Select sources/tabs:** Use the dropdown in the prompt box to choose which spreadsheet tabs Gemini should focus on.
  - Use **Reset to all sources** to re-enable all available sources.

### History Tip

To avoid losing conversation history, insert generated output into the spreadsheet.

You lose Gemini side-panel conversation history when:

- You reload your browser.
- You close and reopen the spreadsheet.
- Your computer goes offline.

### Table Focus Tip

For better analysis when multiple tables exist, focus Gemini’s attention on specific tables.

# Things You Can Ask Gemini in Sheets

---

## Summarize a Spreadsheet

When opening a spreadsheet with data, Gemini may show a summary in the side panel.

You can:

- Click **View more** to expand the summary.
- Use **Good suggestion** or **Bad suggestion** to provide feedback.

### Example prompts

> “Summarize this spreadsheet.”

> “Explain this spreadsheet like I'm 5 years old.”

### Keyboard shortcuts for summarizing

- **Mac:** `⌘ + Ctrl + n`
- **Windows:** `Ctrl + Alt + n`

---

## Create Tables with Gemini

Gemini can generate structured tables directly in Sheets.

### Steps

1. Open a spreadsheet.
2. Click **Ask Gemini**.
3. Select or write a prompt.
4. Press **Enter**.
5. Optionally:
   - Click **Insert** to add the table.
   - Click **Retry** for another version.
   - Add follow-up prompts to refine.
   - Submit feedback.

### Example prompts

> “Create a table for a full day team event.”

> “Create a trip plan for a vacation in a national park.”

> “Create a social media tracker for marketing.”

### Follow-up examples

> “Add another 5 rows of different activities.”

> “Add details on cost.”
