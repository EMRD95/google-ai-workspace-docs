# Create a Notebook in NotebookLM — Summary

**Source:** [NotebookLM Help: Create a notebook in NotebookLM](https://support.google.com/notebooklm/answer/16206563?hl=en)

## Key Excerpts

> **Important:** At this time, the NotebookLM mobile app may have limitations to this feature.

> In NotebookLM, you can create individual notebooks. A notebook is a collection of sources for a specific project.

> **Tip:** Each notebook is independent. NotebookLM can’t access information across multiple notebooks at the same time.

> **Tip:** You can add or remove sources in your notebook by selecting **+ Add** or unselecting individual sources in the “Sources” panel.

> **Tip**: When you have 5+ sources, NotebookLM can auto-label & categorize sources, allowing you to quickly and easily organize your sources by topic.

> **Tip**: Any modifications made within exported Docs or Sheets files do not synchronize with your original NotebookLM content. Sharing permissions do not carry over from NotebookLM to exported files.

## What a Notebook Is

A **NotebookLM notebook** is an independent workspace containing sources for a specific project.

- Each notebook has its own sources, chat, and generated outputs.
- NotebookLM **cannot access information across multiple notebooks at the same time**.
- The mobile app may not fully support all notebook creation features.

---

## Create Your First Notebook

1. Open [NotebookLM](https://notebooklm.google.com/).
2. Select **Create new notebook**.
3. In the pop-up window, choose **Upload a source**.
4. Select or discover the sources you want to add.
5. Start collaborating with NotebookLM using the **Chat** and **Studio** panels.

---

## Main Notebook Panels

### Chat Panel

The **Chat** panel displays a generated summary of your sources.

You can use it to:

- Ask questions about your sources
- Give instructions
- Choose from suggested options
- Interact with NotebookLM based on the uploaded materials

---

### Studio Panel

The **Studio** panel is where NotebookLM generates outputs based on your sources.

Available output types include:

- **Notes**
- **Audio Overviews**
- **Video Overviews**
- **Mind maps**
- **Reports**
- **Data Tables**
- **Flashcards**
- **Quizzes**
- **Slide Decks**
- **Infographics**

NotebookLM may sometimes automatically create starter outputs such as reports, flashcards, infographics, slide decks, audio overviews, or video overviews when sources are first added.

- These auto-generated items are created once.
- They **do not count toward usage limits**.

---

## Reports

In the Studio panel, you can create reports by:

- Creating your own custom report
- Choosing from built-in options:
  - FAQ
  - Study guide
  - Briefing document
  - AI-suggested report type

### Exporting Reports

Open the **three dots menu** next to a generated report to select:

- **Export to Docs**
- **Export to Sheets**, if the report contains data tables

If exported to Sheets:

- Each data table populates its own tab in a Google Sheet.

---

## Data Tables

You can generate a data table directly or customize one.

Customization options include:

- Selecting the **pencil icon**
- Choosing a language
- Using the prompt box to define the table
- Describing rows and columns for a more focused, precise result

### Exporting Data Tables

Open the **three dots menu** next to a generated Data Table and select:

- **Export to Sheets**

Export behavior:

- The data table populates the first tab of a new Google Sheet.
- Citations populate the second tab.

---

## Viewing Prompts Used to Generate Artifacts

You can inspect the prompt used to generate certain Studio artifacts.

For audio overviews, video overviews, flashcards, quizzes, or reports:

- Select **View custom prompt** from the artifact’s three dots menu.

For flashcards, quizzes, and videos:

- A **Show prompt** option is also available next to the title in the viewer.

---

## Adding, Removing, and Organizing Sources

You can manage sources in the **Sources** panel.

Actions include:

- Add sources using **+ Add**
- Remove sources by unselecting individual sources

When a notebook has **5 or more sources**, NotebookLM can automatically label and categorize them by topic.

You can also manually organize sources by:

- Adding a new label
- Renaming labels
- Deleting labels
- Moving sources to different labels
