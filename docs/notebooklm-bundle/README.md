# NotebookLM Import Bundle

Flat collection of all Google AI Workspace documentation, ready for manual import into NotebookLM.

## Structure

- **Individual files** (296 files): One per help article — `{area}__{title-slug}.md`
- **Combined files** (`combined/` folder, 21 files): All articles from one product area merged — `{area}__combined.md`

## How to import

### A. Bulk import by product area (recommended)
1. Open NotebookLM → new notebook
2. Drag-and-drop one or more `combined/*.md` files into Sources
3. Each becomes one source with all articles from that area

### B. Individual files
Drag specific `{area}__*.md` files for focused notebooks

### C. Copy-paste
Open `.md` in text editor, Ctrl+A, paste into NotebookLM

## Naming: `{area}__{slug}.md`

- Double underscore `__` separates product area from title slug
- Max 80 chars, truncated at word boundaries
