# NotebookLM Help Summary: Add or Discover New Sources for Your Notebook

Source: [Google NotebookLM Help — Add or discover new sources for your notebook](https://support.google.com/notebooklm/answer/16215270?hl=en)

---

## Key Excerpts & Important Facts

> **Important:** At this time, the NotebookLM mobile app may have limitations to this feature.

> A source is a copy or auto-synced version of the source document you import or upload to the app.

> When you use NotebookLM, the model uses the sources you upload to answer your questions or complete your requests.

> Each source can contain up to **500,000 words** or up to **200MB** for uploaded files.

> You can include up to **50 sources** for Free users.

> To chat with a specific set of sources in your Notebook, select them individually in the “Source” panel.

> When uploading multiple web URLs, separate links by a space or a new line.

> If the source content is too short, NotebookLM references the entire document without citing individual text from your source.

---

# 1. What Sources Are in NotebookLM

A **source** is either:

- A copy of a document you upload/import, or
- An auto-synced version of a connected source, such as a Google Drive file.

NotebookLM uses your selected sources to answer questions, summarize material, and complete requests within a notebook.

---

# 2. Supported Source Types & Limits

## Supported Source Types

NotebookLM supports importing or uploading the following source types:

### Files & Text

- Copy-pasted text
- Microsoft Word: `.docx`
- Text files: `.txt`
- Markdown: `.md`
- PDFs: `.pdf`
- CSV files: `.csv`
- PowerPoint files: `.pptx`
- ePub files

### Google Drive / Workspace Files

Supported Google Drive files include:

- Google Docs
- Google Slides — up to **100 slides**
- Google Sheets — limited to **100k tokens**
- Supported files from Google Drive generally

### Audio Files

Supported audio sources include MP3 and WAV, among others.

### Images

Supported image file types:

- `avif`
- `bmp`
- `gif`
- `heic`
- `heif`
- `ico`
- `jp2`
- `jpe`
- `jpeg`
- `jpg`
- `png`
- `tif`
- `tiff`
- `webp`

**Note:** Certain types of images may not work as well.

### Web & Online Sources

- Web URLs
- YouTube URLs of public videos
- Gemini Chats

### Gemini Chats

You can chat with your notebooks in Gemini and add them as context to NotebookLM notebooks.

---

## Source Size & Quantity Limits

- Each source may contain up to:
  - **500,000 words**, or
  - **200MB** for uploaded files
- Free users can include up to:
  - **50 sources**

Additional limits and premium features are available through Google’s AI Plans.

---

# 3. Tips for Working with Sources

- Avoid uploading documents you do not have rights to.
- You can create a source by copying and pasting text.
- You can add or edit the title when creating a pasted-text source.
- To chat with only certain sources, select them in the **Source** panel.
- When uploading multiple web URLs, separate each link with:
  - A space, or
  - A new line.

---

# 4. Summarizing Sources

NotebookLM provides two main ways to summarize sources.

## Option 1: Ask in Chat

You can ask NotebookLM to summarize:

- A full source
- Specific sections
- Specific topics
- Information across selected sources

For better results, ask focused questions.

### Better Prompt Example

Instead of:

> "Summarize this source"

Use:

> "What are the key findings about dog training in the 'Dog Training 101' document?"

When multiple sources are selected, include source names in your prompt to help NotebookLM narrow its response.

---

## Option 2: Use the Source Guide

NotebookLM auto-generates a summary of the entire source in the **Source Guide**.

To access it:

1. Open the source viewer on the left side.
2. Select **“To open a source.”**

---

# 5. How to Add Sources

On a computer:

1. Open [NotebookLM](https://notebooklm.google.com/) or Notebooks in Gemini.
2. Select **Add**.
3. Type a research question in the search box to find:
   - Web sources, or
   - Workspace files
4. Select all sources you want to include in your notebook.

---

# 6. Importing Google Drive Files

You can import one or multiple supported Google Drive files when adding sources.

## Auto-Sync Behavior

Sources imported from Google Drive are automatically updated.

- They sync every few minutes.
- Changes to the original document automatically update in NotebookLM.
- You can manually sync by opening the source and clicking:

> ”Click to sync with Google Drive.”

---

## Access Requirements

You can only import Google Drive files if you have:

- **View access or higher**

If you lose access to a Drive file, or the file is deleted:

- The source becomes inaccessible.
- You can no longer view or interact with it in the notebook.
- This applies to both:
  - Owned notebooks
  - Shared notebooks
