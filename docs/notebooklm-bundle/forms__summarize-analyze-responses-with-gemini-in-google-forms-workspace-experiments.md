# Summarize & analyze responses with Gemini in Google Forms (Workspace Experiments) – Comprehensive Guide

This article documents the invite-only **Google Workspace Experiments** feature that brings Gemini AI capabilities into Google Forms for summarizing and analyzing open‑text responses. All details below are extracted from the official Google support page.

---

## ⚠️ Availability & Requirements

- **Invite‑only**: Workspace Experiments is a limited program; not available in all countries ([check availability](https://support.google.com/docs/answer/13607340)).
- **How to check**: Open a new form in Google Forms; if you see “Help me create a form” at the top, you have access.
- **Environment**: Desktop only, English language only ([change language instructions](https://support.google.com/accounts/answer/32047)).
- **Organizational accounts**: If the feature is missing, an admin can enable it for the user.

---

## 📝 Summarizing Responses with Gemini

### Automatic Summary Generation

- **Trigger**: Once a question receives **3 to 200 responses**, a summary block appears automatically for each text question in the **Responses** tab.
- **Where**: `Responses` → look for the summary block under each text question.

### How to Use

1. Open the form and click **Responses**.
2. Summary for each text question is generated automatically (no manual action).
3. **Optional actions**:
   - **Retry**: Click to generate a new summary.
   - **Copy**: Click to copy the summary to clipboard.
   - **Refresh with latest responses**: If new answers arrive while the summary is open, click to update.

> **Important notes**:
> - Summaries **are not saved** – you must copy or regenerate them.
> - Summaries **won’t be generated** for direct questions (e.g., names, addresses).
> - **Quiz responses** are not supported – no summaries for quiz forms.

## 🧩 Generating & Refining Themes from Responses

Extract common themes from open‑text answers and visualize them in Google Sheets.

### Constraints

- **Optimal response count**: **8–200 responses**.
- **If >200 responses**: Theme generation becomes **unavailable** → button grayed out with error message.
- **Underlying technology**: Uses AI functions in Google Sheets ([AI function limits](https://support.google.com/docs/answer/15820999)).

### Step‑by‑Step

1. In the form, go to **Responses**.
2. Scroll to a text question – below the summary, click **See theme percentages** ![](https://storage.googleapis.com/support-kms-prod/HzLKPvJl4iTfF7sJiKIe9GnySygEs9weObLP).
3. **Modify themes** (optional):
   - Click **Modify themes**.
   - **Add** a theme via the Add button ![](https://storage.googleapis.com/support-kms-prod/nFiJUlnD533kXvNsQ79alqMYu5bUwgqgGvhl).
   - **Delete** a theme using Delete category ![](https://storage.googleapis.com/support-kms-prod/fVjpanqSy4DrugRm5ezeJqf30HOAz7jhSWRA).
4. For a deeper dive, click **See theme percentages** again → opens a new Google Sheet containing:
   - A **pie chart** of theme distribution.
   - A **response theme count** table mapping each theme to its number of responses.
   - If an error occurs, use the **Refresh & Generate** button in the Sheets side panel.

> **Tip for extreme counts**: If responses are **≤8 or >500**, the text summary still shows, but the theme percentage button is **grayed out**.

## 💬 Providing Feedback on Generated Outputs

Google encourages feedback to improve accuracy and safety.

1. Below a generated summary, click **Good suggestion** ![](https://storage.googleapis.com/support-kms-prod/mKqkrNZec6ZdDZqqln7wU7EGepuseErh6bLo) or **Bad suggestion** ![](https://storage.googleapis.com/support-kms-prod/hWToSpEzNmnGXO4uonbUrmlmgLRUUzGECUlT).
2. In the pop‑up you can **choose which data** to include (prompts, outputs). Uncheck any you prefer not to share.
3. If you selected **Bad suggestion**, you can then pick the issue type and add extra comments.
4. Click **Submit**.
5. For general feature feedback, use **Help & feedback** at the bottom of the page.
6. Legal issues → [create a request](https://support.google.com/legal/answer/3110420).

## 🚫 Turning Off the Response Summary Feature

- **No per‑feature toggle**: To disable response summaries you must **exit Workspace Experiments entirely**.
- Consequences of exiting:
  - Permanently lose access to **all** Workspace Experiments features ([list](https://support.google.com/docs/answer/13447104#labs_features&labs-ai-docs&labs-ai-gmail)).
  - Cannot rejoin the program later.
- Instructions: [How to turn off Workspace Experiments features](https://support.google.com/docs/answer/13447104#labs-opt-out&exit-labs-docs&exit-labs-gmail).
