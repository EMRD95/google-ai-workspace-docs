## 1. What Gemini in Sheets Can Do

With **Gemini in Google Sheets**, users can:

- **Create tables**
- **Create formulas**
- **Generate data analysis and insights**
- **Build charts and graphs**
- **Summarize files from Drive and emails from Gmail**
- **Generate images**
- **Get help with Google Sheets**
- **Use Google Search from prompts**
- **Use Gems**
- **Solve optimization problems**
- **Perform spreadsheet actions**, including:
  - Apply conditional formatting
  - Create a pivot table
  - Add a dropdown or checkbox
  - Sort data
  - Apply or clear filters
  - Find and replace text
  - Set number formats
  - Insert, delete, or freeze rows and columns
  - Fill a range
  - Format a table
  - Solve optimization tasks

## 2. Availability & Requirements

### Feature Availability

- Requires an **eligible Google Workspace or Google AI plan**.
- Gemini in Sheets is available in **Google Workspace Experiments**, a trusted tester program for users with personal accounts.
- To check whether you have Workspace Experiments access:
  - Open a new presentation in **Google Slides**
  - Look for a **bottom bar**

### Supported Languages

- Gemini in Sheets side panel is available in **some languages**.
- Language availability depends on Google Workspace with Gemini support.

### Important File Compatibility Note

> **Important:** This feature works best with native Google Sheets files. To use Gemini features in an Excel file (.xlsx), select **File** → **Save as Google Sheets**.

## 3. Getting Started with Gemini in Sheets

### Open Gemini

1. Open a spreadsheet in [Google Sheets](https://docs.google.com/spreadsheets/u/0/).
2. At the top right, click **Ask Gemini**.
3. In the side panel:
   - Select a suggested prompt, or
   - Write your own prompt.
4. Press **Enter**.

### Use Suggested Prompts

1. Select a suggestion from the list.
2. To see more options, select **More suggestions**.
3. Replace example text in the prompt box with your own information.
4. Press **Enter**.

### Enter Your Own Prompt

1. Type a prompt in the prompt box.
2. Press **Enter**.

### Optional Controls

- **Undo:** Reverse the last Gemini action.
- **Clear history:** Remove Gemini side panel history.
- **Use specific tabs in a prompt:**
  1. Check the dropdown in the prompt box.
  2. Select only the tabs Gemini should use.
  3. To restore all sources, click **Reset to all sources**.
  4. Enter your prompt.

### Conversation History Warning

To avoid losing history, insert generated output into the spreadsheet.

You lose conversation history when:

- You reload your browser.
- You close and reopen the spreadsheet.
- Your computer goes offline.

### Tip

- For more comprehensive analysis, focus Gemini on specific tables when a sheet contains multiple tables.

## 4. Gemini Side Panel Actions & Controls

| Action | Purpose |
|---|---|
| **Ask Gemini** | Open Gemini. |
| **More options** | Clear recent Gemini history and ask for more suggestions. |
| **Expand** | Make Gemini larger. |
| **Collapse** | Return Gemini to original size. |
| **Close** | Close Gemini. |
| **Clear history** | Remove generated text/images not yet inserted. |
| **More suggestions** | Get more text suggestions. |
| **More image suggestions** | Get more generated image suggestions. |
| **Insert** | Insert generated text into the spreadsheet. |
| **Insert image** | Insert a generated image into the spreadsheet. |

| **View more / View less** | Expand or minimize Gemini’s response. |

| **Undo** | Reverse the last Gemini action. |
| **Export to Docs** | Export Gemini’s response to a new Google Doc. |

## 5. Summarize a Spreadsheet

When opening a spreadsheet with data, Gemini may show a **summary of the spreadsheet** in the side panel.

You can:

- Click **View more** to expand the summary.
- Use **Good suggestion** or **Bad suggestion** to provide feedback.

### Example Prompts

> “Summarize this spreadsheet.”

> “Explain this spreadsheet like I'm 5 years old.”

### Keyboard Shortcuts

To summarize a spreadsheet:

- **Mac:** `⌘ + Ctrl + n`
- **Windows:** `Ctrl + Alt + n`
