# Learn about Google Workspace Studio limits

**Important:** Limits are subject to change.

The Google Workspace Studio limits:

- The number of flows you can create
- How often flows can run per day

## Understand Workspace Studio limits

- **Daily run limit:** This is the maximum number of times all of your flows combined can run in a 24-hour period.
- **Created flows:** You can create a maximum of 25 flows. This includes all flows you create, whether they’re turned on or not.
- **Flows with a Gmail starter:** No more than 25 of your active flows can be started by events in Gmail.
- **Steps per flow:** An flow can have a maximum of 20 steps.

## Learn what happens when you reach the daily run limit

**Important:** The daily run limit resets every 24 hours. Once the limit resets, your flows run again as scheduled.

If you reach your daily run limit, your active flows stop running.

- Error notifications appear in the Workspace Studio side panel of apps, like Gmail or Google Chat.
- After you reach the limit, flows’ attempts to run fail.
- To find which flows are affected, visit the “Activity” tab.

## Learn what to do if you reach a limit

If you frequently hit your daily run limit, you can:

- **Turn off flows:**Disable any flows you don’t currently need.


1. On your computer, go to [studio.workspace.google.com](https://studio.workspace.google.com/).
2. Click **My flows**.
3. Next to a flow, click More ![More](https://lh3.googleusercontent.com/zrEnCyUpV1ojNclw8WBleBtD_fvmPsi_uzzvr-fSGf9fhZ3mm7N9boscCx1P9q1tX8Q=h36)![and then](https://lh3.googleusercontent.com/3_l97rr0GvhSP2XV5OoCkV2ZDTIisAOczrSdzNCBxhIKWrjXjHucxNwocghoUa39gw=h36)**Turn off**.
- **Optimize your flows:**
  - **Frequency:** Check scheduled flows to determine if they can run less often. For example, if real-time updates aren’t critical, update a flow that runs every minute to run every hour.
  - **Filtering:** To prevent flows from unnecessarily running, make your starters more specific.For example, you can add conditions to the starter to only run for emails from a specific sender or a specific word in the subject line instead of from all emails.
