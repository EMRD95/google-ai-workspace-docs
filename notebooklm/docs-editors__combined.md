# Collaborate with Gemini in Google Docs — Summary

**Source:** Google Docs Editors Help  
**URL:** https://support.google.com/docs/answer/14206696?hl=en  
**Platform covered:** Computer

---

## Overview

Gemini in Google Docs helps users write, edit, summarize, refine, and create content directly inside Google Docs. It can also generate images, reference other Workspace files, search selected sources, and use custom or premade Gems for repetitive tasks.

With Gemini in Google Docs, you can:

- Write and edit with Gemini
- Create personalized documents
- Summarize files from Google Drive and emails from Gmail
- Reference other files to edit or refine content
- Create images

---

## Feature Availability

**Important requirements and limitations:**

- Gemini in Docs requires an eligible **Google Workspace** or **Google AI** plan.
- Supported languages vary by feature and location.
- Gemini in Docs is also available through **Google Workspace Experiments**, a trusted tester program for personal accounts.
- To check Workspace Experiments access, open a new Google Slides presentation and look for a bottom bar.

## Supported Languages

You can use the **Gemini in Docs side panel** in some languages.

Google links to additional help pages for:

- Gemini feature and plan availability
- Supported languages for Google Workspace with Gemini
- Language availability for the side panel

## Getting Started with Gemini in Docs

### Open Gemini in a document

1. On your computer, open a document in [Google Docs](https://docs.google.com/document/u/0/).
2. At the top right, click **Ask Gemini**.
3. In the side panel:
   - Select a suggested prompt, or
   - Write your own prompt.
4. Press **Enter**.

### Use a suggested prompt

1. Select a suggestion from the list.
   - To see more options, select **More suggestions**.
2. At the bottom, replace the example text in the prompt box with your own information.
3. Press **Enter**.

### Enter your own prompt

1. At the bottom of the side panel, enter a prompt.
2. Press **Enter**.

### Clear Gemini history

To remove your recent history:

> Click **More options** → **Clear history**

**Tips:**

- To avoid losing generated content, insert Gemini output into the document.
- Gemini conversation history is lost when:
  - You refresh your browser.
  - You close and reopen the document.
  - Your computer goes offline.

## Summarize Text with Gemini

Gemini can summarize an entire document or selected text.

### How to summarize a document

1. Open a Google Docs document.

### After Gemini generates a summary

You can:

- **Add the text to your document:** Click **Insert**.
- **Generate a different version:** Click **Retry**.
- **Send feedback:** Click **Good suggestion** or **Bad suggestion**.

# Create Personalized Documents with Gemini in Google Docs — Summary

**Source:** Google Docs Editors Help  
**URL:** https://support.google.com/docs/answer/15541387?hl=en

---

## Overview

Gemini in Google Docs can generate first drafts that use relevant context from:

- Google Drive files
- Google Chat
- Gmail
- The web

It can also reference existing documents to match:

- **Formatting** — layout, style, and structure
- **Writing style** — tone, sentence structure, and vocabulary complexity

Common use cases include creating:

- Proposals
- Project plans
- Blog posts
- Reports
- Memos
- Templates
- Personalized instructions

---

## Feature Availability

This feature has several access and platform requirements:

- Requires an eligible **Google Workspace** or **Google AI** plan.
- Available for:
  - Eligible Google Workspace with Gemini users
  - Google Workspace Experiments users
  - Google AI Ultra accounts
  - Google AI Pro accounts
- Workspace Experiments is a trusted tester program for personal accounts to try new AI features.
- To check Workspace Experiments access, open a new Google Doc and look for a bottom bar.
- The feature is currently available:
  - **On desktop**
  - **In English only**

> If you don’t have access to personalization when using Help me create, your experience will look different.

## Create Documents with Gemini

To generate a new document draft:

1. On your computer, open a document in [Google Docs](https://docs.google.com/document/).
2. At the bottom bar, enter a prompt describing the document you want to create.
3. Optional: Match an existing document’s formatting, including layout, style, and structure.
4. Optional: Reference an existing document’s writing style.
5. Optional: Add documents as references for the new content.
6. Click **Submit**.
7. Optional: To review sources used, select **Sources**.

---

## Match an Existing Document’s Format

You can use a Google Doc as a template so Gemini generates content matching its:

- Layout
- Style
- Structure

### Steps

1. On an empty Google Doc tab, select **Match doc format** above the bottom bar.
2. In the popup, choose the file to use.
3. Click **Add**.
4. Enter a prompt describing the document you want to create.
5. Press **Enter**.

The new content’s layout, style, and structure will be based on the selected document.

---

## Match a Writing Style

Gemini can reference a Google Drive document to match writing characteristics such as:

- Tone of voice
- Sentence structure
- Vocabulary complexity

### Steps

1. In Google Docs, go to the bottom bar and click **Tools**.
2. Select **Writing style** → **Add doc from Drive**.
3. In the popup, click **Add document**.
4. Select the file to use.
5. Click **Add**.
6. Gemini generates a summary of the writing style it will apply.
7. Click **Confirm**.
8. Optional: If the style summary is not right, select a different file.

### Example Writing Style Summary

> “Formal, objective, and direct. Concise phrasing but avoid acronyms.”

### Tip

If you paste new content into an existing document where Gemini previously used another document to match writing style, select **Match writing style**.

## Reference Other Files to Create Content

You can ask Gemini to use specific files to write, edit, or refine content.

### Example

> “Update the project timeline based on the latest updates from our meeting notes doc.”

### Steps

1. At the bottom bar, click **Sources** → **Add from Drive** or another location.
   - Alternatively, type `@` and search/select a file from the menu.
2. Select one or more files.
3. Click **Add**.

## Choose Where Gemini Can Search for Information

Gemini can proactively search selected locations for content related to your prompt.

Possible source locations include:

- Gmail
- Drive
- Web search
- Other available Workspace sources

You can choose which sources are enabled depending on the task.

## Overview

Google Workspace with Gemini adds AI-assisted features across Workspace apps to help users draft and revise content, summarize information, create images/videos/audio, analyze data, schedule meetings, and automate workflows.

> “With Google Workspace with Gemini, you can use AI-assisted features to draft emails, revise documents, and more.”

## Requirements

To use Google Workspace with Gemini, you must:

- Be **18 or over**
- Have an eligible Google Account:
  - **Work or school account:** Requires an eligible Google Workspace plan
  - **Personal Google Account:** Requires one of:
    - [Google AI Plus](https://support.google.com/googleone/answer/16882689)
    - [Google AI Pro](https://support.google.com/googleone/answer/14534406)
    - [Google AI Ultra](https://support.google.com/googleone/answer/16286513)

### Additional requirements and notes

- Some AI features may not be available in every language.
  - If a feature is unavailable in your language, switch your Google Account to a supported language.
  - See: [Supported languages for Google Workspace with Gemini](https://support.google.com/meet/answer/14925782)
- For **personal Google Accounts in the US**, these Gemini in Gmail features are available **at no charge**:
  - **Help me write**
  - **AI Overviews for email threads**
  - **Suggested Replies**

## How to check whether you have access

For work accounts, Gemini feature availability depends on your Workspace edition.

To check access:

- Look for the **Gemini icon** in Workspace apps.
  - Example: In Gmail or Docs, click **Ask Gemini** in the top right.
- Or [contact your administrator](https://support.google.com/a/answer/6208960).

## What Gemini Can Do in Docs

Gemini can help draft, edit, and refine content in Google Docs using context from:

- Your current document
- Files in Google Drive
- Google Chat
- Gmail
- The web

Key capabilities include:

- **Edit with a prompt**
  - Example:  
    > “Make the introduction more concise and professional.”

- **Refine with quick actions**
  - Highlight text and choose:
    - **Rephrase**
    - **Shorten**
    - **Elaborate**
    - **Bulletize**
    - **Summarize**
    - **More formal**
    - **More casual**

- **Edit using context from another file**
  - Use your own documents as sources to add data, evidence, and citations.

- **Use the writing style of an existing file**
  - Gemini can mirror the writing style of existing documents.

## Write, Edit, or Refine Text with Gemini

Gemini can understand the context of your document, so you don’t always need to select text before prompting it.

### Steps

1. Open a document in [Google Docs](https://docs.google.com/document/u/0/).
2. Click the **Gemini** icon.
3. Enter a prompt in the bottom bar.
4. To target a specific section, mention it in the prompt.

Example:

> “Add evidence to my Research section.”

To refine the entire document, use a broader prompt, such as:

> “Suggest places where I can better defend my argument.”

5. Optional: Add sources for Gemini to reference.
6. Review Gemini’s suggestions directly in the document.

### Applying or Rejecting Suggestions

- To apply changes individually, click **Accept suggestion**.
- To apply all changes, click **Accept all**.
- To reject all changes, click **Reject all**.

You can also manage Gemini conversation history in Workspace.

---

## Edit Text with One Click

Gemini offers quick editing actions through the floating toolbar.

### Steps

1. Open a document in Google Docs.
2. Select the text you want to rewrite.
3. On the floating bar, click **Refine**.
4. Choose one of the suggested actions:
   - **Rephrase**
   - **Shorten**
   - **Elaborate**
   - **More formal**
   - **More casual**
   - **Bulletize**
   - **Summarize**

### Review Options

After Gemini suggests changes:

- Click **Accept suggestion** to apply changes individually.
- Click **Accept all** to apply all changes.
- Click **Reject all** to discard them.

## Reference Other Files to Create Content

You can ask Gemini to use specific files when writing, editing, or refining content.

Example:

> “Update the project timeline based on the latest updates from our meeting notes doc.”

### Add Files as Sources

1. In the bottom bar, click **Sources**.
2. Choose **Add from Drive** or another location.
   - Alternatively, type `@` and search/select a file from the menu.
3. Select one or more files.
4. Click **Add**.

### Select Where Gemini Can Search

You can choose the locations where Gemini can proactively search for content related to your prompt.
